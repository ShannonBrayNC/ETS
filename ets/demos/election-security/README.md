# Election Security RC Demo Data

This directory contains fictional, nonpartisan, non-PII election-adjacent
evidence packets for the ETS RC demo.

The packets demonstrate:

- schema validation;
- privacy classification;
- deterministic packet hashing;
- immutable linkage through `previous_event_hash`;
- simulated signatures for demo review.

ETS is an evidence/audit layer. It is not vote tabulation software and is not
the vote of record.
